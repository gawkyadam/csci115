The Motherboard is the "backbone" of the computer. Every major part has to connect into the motherboard so it can allow them to talk to each other and get the task done. Without it none of the parts can interact. It distributes the power coming from the PSU to the components which gives them the power they need to run and allowing them to communicate as well. It holds all the necessary inputs for the CPU, GPU, RAM, PSU etc., aswell as the inputs for Wi-Fi, peripherals like your mouse, keyboard, Microphone and Audio. Motherboards are built with a BIOS(Basic Input/Output System) installed that contains all the code required to control the keyboard, disk drives and display screen. Red for the CPU, Green for the RAM, Orange for the PSU input to the motherboard, Blue for a storage device, and Yellow for the GPU
