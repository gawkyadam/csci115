The CPU is a computer chip about the size of a pack of gum. It connects to the Motherboard with dozens of its metal prongs at the bottom that carry electrical signals in and out of the chip. That allows it to connect with all the parts to complete tasks. The CPU has four main tasks that it performs. It starts of with fetching memory from the RAM(Random Access Memory), in that memory it collects there are special instructions inside that the CPU is to execute. To get those instructions the CPU decodes the memory holding those instructions, finds out what those instructions are, and execute the instructions using the relevant parts that are needed.
